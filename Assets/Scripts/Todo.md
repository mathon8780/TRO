### 二、基础系统搭建
1. **角色控制系统**
    - 实现角色移动（使用 Rigidbody2D 或 CharacterController）。
    - 添加基本状态（生命值、饥饿、口渴、疲劳等）。

2. **物品与背包系统**
    - 设计可交互物品（可拾取、可使用、可装备）。
    - 实现背包 UI 和物品堆叠逻辑。

3. **世界与地图系统**
    - 设计基于网格（Tilemap）或程序化生成的地图。
    - 实现建筑、门窗、可破坏物体等交互元素。

4. **时间与天气系统**
    - 实现昼夜循环（控制光照、AI 行为）。
    - 添加天气效果（雨、雾等影响能见度或角色状态）。

---

### 三、核心玩法系统开发
1. **僵尸 AI 系统**
    - 使用 NavMesh（3D）或 A* Pathfinding（2D）实现寻路。
    - 设计不同行为状态（巡逻、追击、攻击、听觉/视觉感知）。
    - 优化性能（对象池、LOD、僵尸休眠机制）。

2. **建造与互动系统**
    - 实现家具摆放、墙体建造、 barricade（路障）系统。
    - 添加交互提示（如“按 E 拾取”）。

3. **战斗与伤害系统**
    - 近战/远程攻击判定。
    - 伤害计算（部位伤害、暴击、耐久等）。
    - 死亡与重生机制（或永久死亡）。

4. **保存与加载系统**
    - 序列化角色状态、世界状态、物品位置等。
    - 支持多存档槽位。

### 四、内容填充与优化
1. **美术资源集成**
    - 导入角色动画（Sprite 或 3D 模型）。
    - 制作环境素材（建筑、植被、道具等）。
    - 实现 UI 界面（背包、状态栏、地图等）。

2. **音效与氛围**
    - 添加环境音（风声、僵尸低吼）。
    - 战斗音效、UI 音效。

3. **性能优化**
    - 对象池管理僵尸和物品。
    - 分块加载地图（Chunk-based loading）。
    - 减少 Draw Call（合批、图集）。
